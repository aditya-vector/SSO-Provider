# React Redux Stripe Integration Application POC

##Installation
####npm3 and node v4+ required
``` git clone git@github.com:export-mike/react-redux-boilerplate.git```

```npm install ```

``` npm start ``` (Should start the server at port 3000)

new shell:
``` npm run test:watch ```

### other commands:
test mocha only:
``` npm run mocha:watch ```

test karma only:
``` npm run karma:watch ```

for CI use ```npm test ```

Build:
``` npm run build ```
